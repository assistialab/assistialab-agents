# assistialab-agents

Short description of the project.


## Make targets
- `make venv` – create virtualenv  
- `make install` – install deps + package  
- `make lint` – ruff lint  
- `make fmt` – ruff --fix + black  
- `make test` – run pytest  
- `make ci` – install + lint + test  
- `make clean` – remove venv/build artifacts
